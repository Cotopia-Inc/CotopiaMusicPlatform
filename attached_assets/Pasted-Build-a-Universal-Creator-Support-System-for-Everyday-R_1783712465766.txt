====================================================
UNIVERSAL CREATOR SUPPORT
====================================================

Replace the current idea of "Tip The Creator" with a broader creator support system.

Primary button text:

❤️ Support This Creator

This button should appear anywhere creator content is displayed.

Initially support:

• Songs
• Videos
• Podcasts
• Creator Profile Pages

The database should be designed so future content types can easily support this system without redesigning it.

Future examples:

• Radio Shows
• Interviews
• Live Streams
• Playlists
• Events
• Blogs
• Other creator content
