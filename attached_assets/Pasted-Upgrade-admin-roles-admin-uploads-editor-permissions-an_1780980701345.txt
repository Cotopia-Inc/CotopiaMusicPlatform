Upgrade admin roles, admin uploads, editor permissions, and analytics tracking.

1. MASTER ADMIN ROLE

Add a new top-level role:

master_admin

There should only be one master_admin account.

Seed account:

[admin@cotopia.com](mailto:admin@cotopia.com)
password123
role: master_admin

Only master_admin can assign or change staff roles.

Staff roles:

* master_admin
* admin
* moderator
* editor

Rules:

* master_admin can do everything
* admin can manage content, users, submissions, company hub, settings, uploads
* moderator can manage comments, reports, flagged content, chat moderation
* editor can review songs/videos, listen/watch content, add songs/videos to playlists, recommend content, feature suggestions, and create editorial playlists
* listener, artist, label, and business cannot access staff tools
