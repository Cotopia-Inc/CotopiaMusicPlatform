PROJECT NAME: Cotopia Streaming Platform MVP

OVERVIEW

Build a modern web-based music and video streaming platform for Cotopia. The platform is owned by Cotopia (parent company) and supports artists, labels, businesses, listeners, and administrators.

This is an MVP intended to validate the business model before scaling to enterprise infrastructure.

IMPORTANT

Build the project in a modular and scalable manner.

Do not implement live streaming yet.

Prepare the architecture so live streaming, embed players, APIs, and mobile apps can be added later without major restructuring.

Branding should use placeholder values that can be changed through the admin CMS.
